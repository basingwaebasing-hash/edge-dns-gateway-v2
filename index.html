Privacy & Zero Logs

Traditional DNS

Traditional providers log your IP, timestamps, and every domain you visit — often for advertising.

This DNS Gateway

We operate a strict zero-knowledge policy. Nothing is written to disk or held in memory.

How Anycast Works

Unicast — Single Server

Users far from the server experience high latency — data travels around the globe and back.

Anycast — Nearest Edge

Every user reaches the nearest edge node — near-instant responses regardless of location.

Smart Ad Blocking

The Filter Layer

Ad and tracker requests are intercepted at the DNS level and returned as null — never reaching your device.

Filter Lists Used

  • AdGuard DNS Filter
  • ABPVN (Vietnam optimized)
  • 1Hosts (Lite)

Multiple premium blocklists are combined and updated regularly for comprehensive coverage.

Advanced: ECS Customization

  • 1Open C:\dns-bibica-net-doh\dnsproxy\config.yaml to see 5 pre-configured ECS lines:
    • #edns: true #edns-addr: "1.32.226.0" # Japan #edns-addr: "212.165.2.0" # Sing #edns-addr: "2.20.40.0" # Hong Kong #edns-addr: "3.2.88.0" # Thai Lan
  • 2Uncomment (remove #) to select your preferred ECS. The final configuration will look like this:
    • listen-addrs: - 127.0.0.1 listen-ports: - 53 upstream: - h3://doh.bibica.net/dns-query - https://doh.bibica.net/dns-query bootstrap: - 1.1.1.1:53 - 8.8.8.8:53 cache: false edns: true edns-addr: "1.32.226.255"
  • 3Run C:\dns-bibica-net-doh\dns-bibica-net-restart.bat to apply changes.
